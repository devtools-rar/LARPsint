# TracePoint OSINT Tool

A high-performance, browser-based OSINT dashboard for tracking digital footprints across public data leaks.

## Features
- **Discord ID Lookup**: Resolves Discord Snowflake IDs to user profiles (username, avatar, creation date).
- **Phone Number Analysis**: Identifies region and correlates with known historic breaches (Facebook, Ledger, etc.).
- **Real-time Terminal**: Visual feedback of the scanning process with simulated intelligence correlation.
- **Cyber-Security Aesthetic**: Dark mode, neon accents, and smooth animations.

## How to Use
1. Open `index.html` in any modern web browser.
2. Select your search type (Phone or Discord).
3. Enter the target data.
4. Click **Initiate Scan**.

## Data Sources
- **Discord Data**: Queried via `discordlookup.mesalytic.moe`.
- **Leak Intelligence**: Historical data correlation based on common breach indices. 
- *Note: For full real-time breach results, integration with paid APIs like LeakCheck or DeHashed is recommended.*

## Safety & Ethics
This tool is designed for blue-team security professionals and authorized OSINT investigations. Do not use for doxxing or illegal activities.
